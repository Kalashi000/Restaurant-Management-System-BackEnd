This is the core backend logic for the **Flavor Hub** Restaurant Management System. It handles data processing, user authentication, and menu management to support the frontend application.

---

##  Architecture Plan
This backend is designed using a **3-Tier Architecture** to ensure scalability and organized data flow:
1. **Controller Layer:** Handles incoming API requests from the Frontend.
2. **Service Layer:** Contains business logic and data processing.
3. **Repository Layer:** Manages communication with the MySQL database.

---

##  Planned Tech Stack
* **Language:** Java (JDK 17+)
* **Framework:** Spring Boot / Servlets
* **Database:** MySQL (Relational Database)
* **API Testing:** Postman
* **Build Tool:** Maven

---

##  Core Functionalities (Planned)
* **User Management:** Secure login and registration for customers and admins.
* **Order Processing:** API endpoints to receive and store food orders.
* **Menu API:** Dynamic data retrieval for the frontend menu categories.
* **Admin Dashboard Logic:** Backend support for updating prices and adding new dishes.
